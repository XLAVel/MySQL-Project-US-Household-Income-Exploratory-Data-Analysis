📊 US Household Income Analysis (MySQL Project)

This project explores the US Household Income dataset using MySQL to perform data cleaning, transformation, exploratory data analysis (EDA), and automation. The goal is to uncover meaningful insights about income distribution across the United States, including geographic trends and potential anomalies within the data.

Through SQL-driven analysis, this project highlights patterns such as income disparities between states and cities, outliers within the dataset, and broader economic trends that emerge from the data.

🧠 Project Objectives

Clean and transform raw income data using MySQL

Perform exploratory data analysis (EDA) to identify patterns and anomalies

Analyze income distribution across states and cities

Identify high-income and low-income regions

Demonstrate SQL techniques for data preparation and analytical querying
